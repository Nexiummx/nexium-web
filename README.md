# nexium-web

Sitio corporativo de [Nexium](https://www.nexiummx.com). Next.js 14 (App Router), TypeScript, CSS Modules.

## Desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Variables de entorno

Copiar `.env.example` a `.env.local` y completar `NEXT_PUBLIC_GA_ID` y `NEXT_PUBLIC_META_PIXEL_ID` si aplica.

## Deploy

Vercel con dominio `nexiummx.com` (ver `nexium-web.md` en el repo padre para el plan completo).
